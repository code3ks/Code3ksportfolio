## 📄 CV Download Setup

To enable CV download functionality:

### Option 1: Use Your Own CV
1. Save your CV as `Sadiq_Muhammed_CV.pdf`
2. Place it in the root directory of this project
3. Push to GitHub

### Option 2: Convert the Included CV Template
1. Open `cv.html` in your browser
2. Press `Ctrl+P` (Windows) or `Cmd+P` (Mac)
3. Select "Save as PDF"
4. Save as `Sadiq_Muhammed_CV.pdf` in the root directory
5. Push to GitHub
